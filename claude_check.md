# 有機分類帽 - 遊戲評估清單 (Claude Check)

> 評估對象：哈利波特風格有機化學官能基分類遊戲
> 模式：自我修煉 / 競速挑戰 / 巫師對決
> 評估日期：2026-05-11

---

## 一、好不好玩（遊戲性 Playability）

### 🎯 核心循環與回饋
- [ ] 答對 / 答錯的「立即回饋」是否夠爽？音效、動畫、分數跳動的節奏感
- [ ] Combo 系統是否真的有「越打越爽」的感覺？x2、x5、x10 有沒有視覺差異化
- [ ] HP 條（反應產率）的設計：答對回血 +5、答錯 -20 的數值是否平衡？太容易死還是太難死？

### 🧗 難度曲線
- [ ] Level 1→6 的難度是否漸進合理？玩家會卡關還是太順？
- [ ] 競速模式 60 秒 + 1 秒冷卻懲罰，會不會讓答錯後失速感太重？
- [ ] Level 99 全英文挑戰是「賞給高手的禮物」還是「沒人想碰的牆」？

### 🔁 重玩動機（Replayability）
- [ ] 通關後有沒有「想再來一次」的理由？（最高分、解鎖、稱號…目前似乎沒有紀錄系統）
- [ ] 隨機抽題池是否夠大？玩兩三次後會不會看到一樣的題

### ⚔️ 對決模式
- [ ] 共用題目時，誰先按誰得分，會不會變成「比手速不比化學」？
- [ ] P1 / P2 視覺區分（顏色、攻擊方向）是否清楚

### 🎨 沉浸感
- [ ] 哈利波特 + 化學的世界觀統一度（「分類帽」「巫師對決」「試管破了」這些梗有沒有讓人會心一笑）
- [ ] 美術風格（羊皮紙、魔法石按鈕）一致性

## 二、過程邏輯是否合理（Logic & Correctness）

### 📚 教學邏輯
- [ ] **分類正確性問題**：[data(organic).js:40-44](data(organic).js#L40-L44) 把 phenol / benzyl_alcohol / benzaldehyde 都歸成 `confused` 類，學生答題時看到正確答案是「易混淆」會不會反而困惑？應該是要答它真正的官能基才對
- [ ] Level 編排是否符合教學序列（碳氫→單鍵氧→雙鍵氧→雙氧→雜原子）✅ 這個排得不錯
- [ ] **前後分類不一致**：Level 99 把 phenol 對應到 `CAT_PHENOL`，但 Level 6 卻歸 `confused`

### 🎮 遊戲機制邏輯
- [ ] [game(organic).js:227](game(organic).js#L227) 用 `Math.random()` 抽題，可能短時間內重複同題
- [ ] **干擾選項偏弱**：[game(organic).js:239-244](game(organic).js#L239-L244) 純隨機抽，沒有「故意挑相似官能基」當干擾，難度偏低
- [ ] 練習模式 `timeLeft = 0` 但仍會跑 `gameLoop`？實際看程式 [game(organic).js:116](game(organic).js#L116) 練習模式沒啟動 timer ✅ OK
- [ ] 答錯 1 秒鎖定 + 扣 20 HP 同時觸發，連扣 5 次就死，配 1 秒冷卻可能造成「眼睜睜看自己被連扣」的挫折

### 🐛 一致性 / 邊界
- [ ] HP < 0 才結束，但對決模式雙方同時歸零的處理？
- [ ] 對決模式答對「立即換題」，對方還沒看完就換掉了 → 是 feature 還是 bug？
- [ ] 沒有「題目庫掃過一輪」機制，可能同題連抽三次

## 三、建議的測試方法

| 項目 | 怎麼測 |
|---|---|
| 好玩度 | 找 2-3 個高中學生實際玩 5 分鐘，觀察表情和「還想再玩嗎」 |
| 教學效果 | 玩前 / 玩後做個 5 題小測驗，看分數有沒有提升 |
| 分類邏輯 | 請化學老師掃過 `category` 欄位 |
| 數值平衡 | 自己用「全對」「答對答錯交錯」「全錯」三種策略各跑一次，看分數 / HP 曲線 |

## 四、優先改善建議（依影響程度排序）

1. **🔴 高**：修正 `confused` 類別的分類邏輯（Level 6 vs Level 99 不一致）
2. **🟡 中**：干擾選項改為「同類或相似官能基」優先抽取，提升鑑別度
3. **🟡 中**：加入抽題不重複機制（避免同一輪重複出題）
4. **🟢 低**：加入最高分紀錄 / 稱號系統提升重玩動機
5. **🟢 低**：對決模式考慮「兩人各自題目」或「答對才換題給對方追」的變體
