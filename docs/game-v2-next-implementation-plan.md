### 3. 中高級選項池改善

狀態：**done**

需求：

- 目前中高級如果從所有化合物取選項，誘答力不足。
- 目標：答案同類 / 相似類 2 個，其他類 1 個。

例：

- 答案：甲醇
- 選項：乙醚、甲醛、甲苯

「相似類」定義：

- 第一版先沿用 level 一開始已定義的 family / 類別。
- 例如涵氧家族內互為相似類。

性質：

- 題目選項產生邏輯
- 可能涉及 `question-engine.js`、`data.js` 或 level data selection
- 不是 CSS
- 不應混入教學關卡 commit

建議實作原則：

- 不新增一套複雜分類系統。
- 先用既有 family / level 定義作為相似類來源。
- 若同類候選不足，再 fallback 到其他類。
- 不要讓答案本身重複出現在 distractors。

驗證：

- 中高級題目至少優先抽 2 個同 family distractors。
- 同 family 不足時仍能正常生成 4 個選項。
- beginner 行為若原本不同，不應被意外改壞。
