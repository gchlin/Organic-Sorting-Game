# 分類帽提示詞（COACH_LINES）

> 來源：`game.js` 第 135-141 行  
> 用途：練習模式中，分類帽在各個情景下的台詞  
> **注意**：修改時只改引號內的文字，不要破壞引號、方括號、逗號的配對

---

## 1. 介紹（game 開始時）

**欄位名稱**：`intro`

```javascript
intro: "先看結構裡最醒目的官能基，再選分類。需要提示就按下方的「💡 提示」。"
```

**使用時機**：
- 進入練習模式時
- 關閉提示模式時（回到預設說明）

**語氣**：直接、無廢話、強調方法

---

## 2. 答對反饋（隨機選一個）

**欄位名稱**：`correct`

```javascript
correct: [
  "答對了，這頂帽子認同你的判斷。",
  "很好，官能基抓得準。",
  "分類成功，下一個結構。"
]
```

**使用時機**：玩家答對時（隨機選一個）

**語氣**：認可、簡潔、不過度讚美（符合分類帽的冷漠風格）

---

## 3. 連錯鼓勵（連錯 3 題以上時）

**欄位名稱**：`streak`

```javascript
streak: "連錯幾題時，先找氧、氮、鹵素，再看有沒有 C=O 或苯環。"
```

**使用時機**：玩家連錯 3 題時，接在「這個結構是『XXX』」的後面

**完整顯示範例**：
```
「乙醇」是「醇」—– –OH 接在飽和（sp³）碳上。
連錯幾題時，先找氧、氮、鹵素，再看有沒有 C=O 或苯環。
```

**語氣**：講重點、給救星、簡單判斷流程

---

## 4. 判斷流程指南（開啟提示時）

**欄位名稱**：`guide`

```javascript
guide: "判斷順序：① 有苯環嗎？②有 C=O 嗎—接 H 是醛、夾在中間是酮、配 -OH 是羧酸、配 -O-碳 是酯。③ 有 -OH 嗎—接苯環是酚、接飽和碳是醇；C-O-C 是醚。④ 有 N 是胺、有 F/Cl/Br/I 是鹵化物、有 C=C 是烯、C≡C 是炔；都沒有就是烷。"
```

**使用時機**：
- 玩家按「💡 提示」開啟提示模式時
- 下一題顯示時（一直保持）
- 玩家答錯但連錯少於 3 題時

**結構**：4 層順序判斷
1. **苯環** → 分支出「芳香烴」或「酚」（看有沒有官能基）
2. **C=O（羰基）** → 分支出「醛」「酮」「羧酸」「酯」
3. **-OH（羥基）** → 分支出「酚」「醇」，或「醚」（C-O-C）
4. **其他** → 「胺」「鹵化物」「烯」「炔」或「烷」

**語氣**：教練式、有邏輯、由粗到細

## 📝 提示詞修改指南

### 修改時保持一致的風格

✅ **可以修改**（只改引號內的文字）：
- 措辭精準度（例如「官能基」→「官能基群」）
- 教學語氣（嚴厲 ↔ 溫和）
- 簡化複雜說明

❌ **不要改**：
- 引號、方括號、逗號的配對
- JavaScript 物件結構
- 欄位名稱（`intro`, `correct`, `streak`, `guide`）

### 修改後的驗證清單

- [ ] 新文案在 NotebookLM 與化學老師同意
- [ ] 語氣與劇情對話（`story.js`）保持一致
- [ ] 沒有超過原文長度太多（UI 空間有限）
- [ ] 檢查有沒有遺漏「易混淆點」（例如苯酚 vs 苯甲醇）
