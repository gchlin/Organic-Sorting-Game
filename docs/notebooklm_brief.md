## 二、需求項目（逐項貼，或一次全貼）

### 項目 A — 13 類官能基的「辨識重點 + 常見陷阱」（最優先 ⭐）

> 用途：練習模式答錯時，分類帽會講一句「你怎麼看出它是哪一類」。

請針對以下 13 類，各寫：
- **辨識重點**：1～2 句，「在結構式上要看哪裡、看到什麼就是它」。
- **常見陷阱**（如果有）：1 句，「最容易跟它搞混的是什麼、怎麼區分」。

13 類（中文名 / 英文名 / 課本對應的官能基或結構）：

1. 烷烴 alkane（只有 C–C 單鍵、C–H）
2. 烯烴 alkene（C=C 雙鍵）
3. 炔烴 alkyne（C≡C 參鍵）
4. 醇 alcohol（–OH 接在飽和碳上）
5. 醚 ether（C–O–C，氧夾在兩個碳之間）
6. 醛 aldehyde（–CHO，羰基在碳鏈末端）
7. 酮 ketone（羰基夾在兩個碳之間）
8. 羧酸 carboxylic acid（–COOH）
9. 酯 ester（–COO–C）
10. 胺 amine（–NH₂ / –NHR / –NR₂，氮接碳、沒有羰基）
11. 芳香烴 aromatic hydrocarbon（含苯環，環上只接碳氫基）
12. 有機鹵化物 halide（C–X，X = F/Cl/Br/I）
13. 酚 phenol（–OH 直接接在苯環的碳上）

**參考用的現有草稿**（請你改寫得更「對零基礎學生友善」、更貼近課本說法）：

| 官能基 | 現有草稿 |
|---|---|
| 烷 | 只有 C–C 單鍵和 C–H，沒有任何含氧/氮/鹵的官能基 |
| 烯 | 含有 C=C 雙鍵 |
| 炔 | 含有 C≡C 參鍵 |
| 醇 | –OH 接在飽和（sp³）碳上 |
| 醚 | C–O–C，氧夾在兩個碳之間（沒有 OH、也沒有 C=O）|
| 醛 | –CHO：羰基 C=O 在碳鏈末端，旁邊接一個 H |
| 酮 | 羰基 C=O 夾在兩個碳之間 |
| 羧酸 | –COOH：同一個碳上同時有 C=O 和 –OH |
| 酯 | –COO–：像羧酸但 –OH 被換成 –O–碳 |
| 胺 | 含 –NH₂ / –NHR / –NR₂（氮接碳，沒有羰基）|
| 芳香烴 | 含苯環，且苯環上只接烷基/乙烯基等碳氫基 |
| 鹵化物 | 含 C–X（X = F、Cl、Br、I）|
| 酚 | –OH 直接接在苯環的碳上（接 –CH₂– 之類則算醇）|

**輸出格式**（請嚴格照這個，方便我直接複製）：

```
| 官能基 | 辨識重點 | 常見陷阱 |
|---|---|---|
| 烷烴 | …（你的課本式版本）… | …（或留空）… |
| 烯烴 | … | … |
| …（13 列）… |
```

> 加分項：如果課本裡有「按順序檢查」的判斷流程（例如先找苯環 → 再看有沒有 C=O 羰基、在末端還是中間 → 再看 –OH 接在哪 → 最後看有沒有氮/鹵素），請另外寫一段「課本式的辨識流程（給學生按步驟用）」。

### 項目 B — 每關「分類帽新手教學」投影片（2～4 頁）⭐

> 用途：玩家進入每一關之前，分類帽會先放 2～4 頁教學（每頁會配一張該官能基的「純結構示意圖」）。這是要教「從沒學過的人」入門用的，所以要最白話。

請針對下面每一關，寫一組 **2～4 頁**的投影片，每頁格式：
- **標題**：5～10 字
- **內文**：2～4 句，白話、具體；可以直接對學生說話的口吻（不用刻意毒舌，這是教學頁）

每頁建議涵蓋（不一定每關都要 4 頁，2 頁也行）：
①「這一關要認的官能基長什麼樣」（描述結構特徵——會配示意圖）→ ②「怎麼一眼認出它」（看哪裡）→ ③「最容易搞錯的地方」→ ④「生活中的例子 / 為什麼要學」（課本生活連結）。

各關涵蓋的官能基：

| 關卡 | 主題 | 要教的官能基 |
|---|---|---|
| Level 1 | 碳氫骨架 | 烷烴、烯烴、炔烴、芳香烴（苯環）|
| Level 2 | 單鍵氧家族 | 醇、醚 |
| Level 3 | 雙鍵氧家族 | 醛、酮（羰基的位置）|
| Level 4 | 雙氧複合 | 羧酸、酯 |
| Level 5 | 雜原子與鹵素 | 胺、有機鹵化物 |
| Level 6 | 綜合題（複習）| 1～5 全部混；特別強調「酚」自成一類、苯甲醇是醇不是酚 |
| 「龜殼與它的產地」（新關）| 苯環陷阱專練 | 苯環上接不同官能基 → 苯酚（酚）、苯甲醇（醇）、苯甲醛（醛）、苯甲酸（羧酸）、苯胺（胺）、甲苯/二甲苯（芳香烴）、氯苯（鹵化物）、苯乙酮（酮）、苯甲酸甲酯（酯）、苯甲醚（醚）。重點：看到苯環不要急著選「酚」，要先看苯環上接的是什麼 |
| Level 99 | 資優全英挑戰 | 同上 13 類，但要用英文類別名（Alkane / Alkene / … / Carboxylic Acid / Ester / Phenol …）；這一頁順便列課本上這些英文學名的對照 |

**輸出格式**：

```
## Level 1 — 碳氫骨架
### 第 1 頁：標題
內文…
### 第 2 頁：標題
內文…
（最多 4 頁）

## Level 2 — 單鍵氧家族
…
```

## 三、拿回文案之後怎麼用

1. **項目 A** → 填進 `化學審查清單.md §3`（給老師審），審完放進 `data.js`（未來會有「為什麼提示」欄位）。
2. **項目 B** → 進 `化學審查清單.md`（給老師審），審完做成「每關新手教學」的資料（`game.js` modal 要先擴充成可放圖片）。
3. **項目 C** → 微調 `story.js` 對應關卡的對話陣列。
4. 任何 NotebookLM 標「（待查課本）」或老師覺得不對的，回頭再問 / 再改一行字即可。
